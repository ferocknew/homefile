== Description ==

An elegant, versatile and browser-safe theme with a power-packed set of options. It has 19 widget areas, one-column, two-column and three-column
fixed-width and flexible-width formats, 10 pre-defined templates, 17 pre-defined color schemes, two customizable multi-level drop-down menus,
featured posts, a magazine layout, tabbed sidebars, widgets for Twitter, Social Networks and Google Translator, translations in many languages
and RTL language support. WP 3.0 Menus, Custom Post Types and Custom Taxonomies are integrated. A BuddyPress support pack is available as a
plugin for smooth BuddyPress integration. Support forum at http://www.aquoid.com/forum.

Translations for the theme are available at http://aquoid.com/news/themes/suffusion/translating-suffusion/. The back-end of the theme is
not currently translated. However a plugin will be made available soon to facilitate translations of the same.

The theme supports custom headers through its admin panels: Suffusion Options -> Skinning -> Header

The theme has special support for the Chat, Status, Audio and Gallery post formats, in addition to the Standard post format. For the Audio
post format, the standalone version of the <a href='http://wpaudioplayer.com'>WP Audio Player</a> is used. The source for the SWF file can be found
here: http://tools.assembla.com/1pixelout/browser/audio-player/trunk/source. The code for WP Audio Player (including the SWF, FLA and JS files)
is distributed under the MIT license as per its <a href='http://wpaudioplayer.com/license/'>License</a> page.

The files titled iconset-*.png and post-formats-*.png use some icons released by StudioPress under GPL as outlined in the downloaded packages from
<a href='http://www.studiopress.com/download/icon-set-bloggers.zip'>here</a> and <a href='http://www.studiopress.com/download/post-format-icons.zip'>here</a>
respectively.

The "Follow Me" widget uses icons from the following sources:

*	Google+ icon set from Theme.fm, distributed under GPL v2 (http://theme.fm/2011/08/12-gorgeous-google-icons-for-your-website-1280/)
*	"Web Social" by Narjis Naqvi - Free for personal and commercial use (http://www.iconspedia.com/pack/web-social-2021/)

*	ElegantThemes media icons from ElegantThemes.com - Dual licensed under MIT and GPL v3.0 (http://www.elegantthemes.com/blog/resources/free-social-media-icon-set)

Footer credits for the theme can be edited by changing the file called "custom/site-footer.php". It is advisable to create
a child theme and create a folder called "custom" there, then copy site-footer.php into it.

== Installation ==

You can install the theme through the WordPress installer under <strong>Themes &rarr; Install themes</strong> by searching for it.
Alternatively you can download the file from here, unzip it and move the unzipped contents to the <code>wp-content/themes</code> folder
of your WordPress installation. You will then be able to activate the theme.

== Frequently Asked Questions ==

= Where can I get support for the theme? =
Support is available at the <a href='http://www.aquoid.com/forum'>theme's support forum</a>.

= Does Suffusion work with BuddyPress? =
Yes. There is a plugin called <a href='http://wordpress.org/extend/plugins/suffusion-buddypress-pack/'>Suffusion BuddyPress Pack</a>,
more information about which can be found <a href='http://www.aquoid.com/news/plugins/suffusion-buddypress-pack/'>here</a>.

= What languages is Suffusion available in? =
There are several translations distributed <a href='http://www.aquoid.com/news/themes/suffusion/translating-suffusion/'>here</a>.

= Where can I get more information about the sidebars/widget areas in Suffusion? =
Suffusion has multiple widget areas at different locations. For a diagram of the sidebar layout go to Suffusion Options &rarr; Sidebars &amp; Widgets &rarr; Sidebar Layout.
