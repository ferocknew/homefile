综述：美国空军无人机研制近况

浏览次数：2 发表时间：2001/6/2 关键字：美国空军 来源：每日防务快讯网

    [《今日美国》2001年5月报道] 诺斯罗普·格鲁门公司为美国空军生产的、可留空36小时的RQ-4A"环球鹰"无人机最近完成了23小时、8600英里跨太平洋不间断飞行后，似乎飞行器的发展又跨入一个新时代。无人机的研制正方兴未艾，美国正在计划研制中的侦察和攻击无人机有以下几个重点型号：

　　"环球鹰"：计划到2021年制造63架，单机售价约1200万美元，比普通侦察机平均价低得多(EP-3螺旋桨侦察机为8000万美元)。目前，"环球鹰"已获1.615亿美元预算。

　　"捕食者"：正在进行可携导弹的重新设计。2001年获8500万美元预算，预计单机售价320万美元。

　　"黑寡妇"：为螺旋桨驱动、比手掌稍大的微型飞行器(MAV)，正由位于加利福尼亚锡米瓦利的美国航空环境公司(Aero Environment)为美国国防部研制。该机用锂电池作动力，用于地面部队发现和识别方圆几英里内的敌方武装。

　　目前，"黑寡妇"可以43英里/小时的速度飞行20分钟，研制者希望提高到1小时。机上的机载雷达可将活动图像传输给接收机，这对想了解前方战场情况的连队指挥官来说是无价信息。

　　"Helios"无人机：美国航空环境公司正同NASA一道研制，计划数月后飞行。
